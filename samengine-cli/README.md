# Samengine CLI

This package contains 2 CLI Tools which can used with samengine
and although other projects

Run them by typing this:

```sh
npx samengine-cli-linksaver --help
npx samengine-cli-buildinfos --help
```

## INFO

This package uses other Version numbers than **samengine** and
**samengine-build** because it does not depend on them!
